# StreamVault Dashboard

Svelte 5 frontend for StreamVault — manages media queue, library, and settings.

## Development

```bash
npm install
npm run dev     # Vite dev server, proxies /api to localhost:8080
```

## Build

```bash
npm run build   # Output: dist/
```

## Stack

- Svelte 5 with TypeScript
- Vite
- SSE real-time events
- Glassmorphism UI
